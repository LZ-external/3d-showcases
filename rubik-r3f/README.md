# Rotating Rubik’s Cube – React Three Fiber

Small example built with React and Vite to explore Three.js library / React Three Fiber.

## Goal
- Learn basic Three.js concepts in React
- Understand meshes, materials, lights, and animation
- Experiment with 3D scenes in a simple setup

## Features
- Rubik’s Cube made of 27 small cubes
- Colored faces (stickers)
- Continuous rotation
- Mouse controls to rotate and zoom

## Tech Stack
- React
- Vite
- Three.js
- @react-three/fiber
- @react-three/drei

## Run the project

Install dependencies:
npm install

Start the dev server:
npm run dev

Open the local URL shown in the terminal.
